This release focuses on a small set of fixes around timer flow and settings behavior.

- Picking a historical day now switches the main timer back to the current business day and starts the live session.
- Minimizing the window now behaves like a normal minimize instead of sending the app to tray.
- Closing the window still sends the app to tray.
- AM/PM is now available as a separate time-format option.
- Selected settings buttons now fill with orange instead of only showing an underline.
