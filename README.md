# Minimal Worktime Tracker

Минималистичное desktop-приложение на Electron для учета рабочего времени.

## Что умеет

- Таймер рабочего времени
- Переключение перерыва одним нажатием
- Календарь с прогрессом `часы / 6 ч` по дням
- Серия рабочих дней подряд с пропуском выходных
- Настройка выходных (можно отметить любой день)
- Скрытие в трей при закрытии окна

## Запуск

```bash
npm install
npm start
```

После запуска закройте окно через `X`, и приложение останется в трее.

## Сборка

```bash
npm run dist
```

Артефакты Windows-сборки появятся в `release/`:

- `*.exe` installer для установки
- `*.exe` portable-версия для запуска без установки
