This release tightens a few day-to-day behaviors in the timer and settings screens.

- Picking a historical day now snaps the main timer back to the current business day and starts the live session.
- Minimizing the window now behaves like a normal minimize; only closing the window sends the app to tray.
- AM/PM is now available as its own time-format option.
- Selected settings buttons now fill with orange instead of using only an underline.
